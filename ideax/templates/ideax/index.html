Ideia X compartilhando ideias!

Com o Ideia X você tem um canal aberto para a inovação, recebendo ideias de todas as partes, avaliando e fazendo-as acontecer!

Lance suas ideias para todos!

O Ideia X é o canal para divulgar e compartilhar suas ideias, permitindo que seus pares possam contribuir, discutir e colaborar para o crescimento dela.

Da mesma forma, você pode ver as outras ideias postadas, interagindo com sua opinião, perguntando e opinando para melhorar ainda mais outras ideias.

Funcionalidades

Cadastro de ideias

Todos dentro da empresa ou projeto podem colaborar. A inovação poderá vir de todos.

Gestão de ideias

As ideias podem ser organizadas, amadurecidas e avaliadas, permitindo um fluxo para a gestão da inovação.

Integração com outras ferramentas

Através das APIs é possível integrar com outras ferramentas. Integrando com uma ferramenta de gestão de projetos é possível criar um projeto a partir de uma ideia cadastrada no Ideia X.
